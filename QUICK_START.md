- Close when ready

---

### 4️⃣ **Scan for Threats** 🔍
- Click **"Detect Threats"** from dashboard
- Click **"Test Threat"** button (right side panel)
- Form auto-fills with threat data
- Click **"🔍 Analyze Network Traffic"**
- Wait 2 seconds for AI analysis

---

### 5️⃣ **See Results** 📊
- Threat detected! (Red alert screen)
- Shows confidence score (e.g., 95%)
- Click **"🔧 Resolve This Threat Now"**

---

### 6️⃣ **Auto-Fix the Threat** 🛡️
- Taken to ATR Dashboard
- Review suggested actions
- Click **"🛡️ Execute Response"**
- Watch 6-step automatic resolution:
  1. 🔍 Analyzing Threat
  2. 🔒 Isolating Source
  3. 🛡️ Applying Countermeasures
  4. 📊 Updating Rules
  5. ✅ Verifying Protection
  6. 📝 Logging Response
- Success! 🎉 (Confetti animation)

---

### 7️⃣ **Generate Report** 📄
- Return to Dashboard
- Click **"Generate Report"**
- Choose **"Comprehensive Report"**
- File downloads automatically! ✅

## 🆘 TROUBLESHOOTING

**Q: Login doesn't work?**
A: Make sure you're using `demo` / `demo123` (lowercase)

**Q: Test Threat button not showing?**
A: It's in the right panel under "Quick Test Presets"

**Q: ATR doesn't execute?**
A: Click "🛡️ Execute Response" button after threat loads

**Q: Report doesn't download?**
A: Check your browser's download folder for .txt file

**Q: Animations not smooth?**
A: Refresh the page (Ctrl+F5) to reload CSS
